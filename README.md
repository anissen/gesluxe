Gesluxe
=======
Inspired by Gestouch ActionScript 3 gesture recognition library

##Supported gestures:
 * Tap
 * Pan
 * Long press
 * Swipe
 * Zoom
 * Rotation
 * Transform


It should work on every target supported by Luxe. Tested and working on CPP (Window, Mac, iOS, Android) and Web
