Gesluxe
=======
Basically is a fork of https://github.com/fljot/Gestouch

##Supported gestures:
 * Tap
 * Pan
 * Long press
 * Swipe
 * Zoom
 * Rotation
 * Transform


It may work on every target supported by Luxe. Only tested in a windows machine (cpp target) and android (tablet and mobile).
